
# Preferences Demo

A demo application to demonstrate implementing a standard preferences window in native macOS applications at [macOS native Symposium #02](http://macos-native.github.io).


## Environment

- System: macOS 10.13 or later
- Swift: Swift 4.2
- Xcode: Xcode 10.1
